Domain Specific Language for HTML.

HTML()

declare()void

Defines the structure and content of the HTML document or fragment.

This abstract method serves as a blueprint for subclasses to implement and define the specific elements, attributes, and hierarchy of the HTML structure. It is typically invoked to initialize and build the desired HTML content.

Subclasses should override this method to declare the HTML nodes and their relationships using the provided methods such as $, attr, and others.

$(HTMLhtml)void

HTMLhtml

The HTML instance to be embedded. Its structure will be integrated into the current context.

Embeds the structure of another HTML instance into the current HTML document. This method allows the reuse and composition of existing HTML definitions.

The declare() method of the provided HTML instance is invoked to generate its structure. The resulting XML nodes are then added to the current tree.

attr(Objectname)Consumer

Objectname

The name of the attribute to be added. If null or empty, no attribute will be added.
Consumer

A Consumer that can be used to set the specified attribute on an HTML node.

Creates a Consumer that sets an attribute on an HTML element.

This method allows you to define an attribute for an HTML element by providing its name. The attribute will be added to the target HTML node when the returned Consumer is invoked.

If the provided name is null or an empty string, the attribute will not be added.

Objectname

The name of the attribute to be added. If null or empty, no attribute will be added.
Objectvalue

The value to be assigned to the attribute. If null, the attribute will still be added, but its value will be represented as &quot;null&quot;.
Consumerkiss.XML

A Consumer that can be used to set the specified attribute and value on an HTML node.

Creates a Consumer that sets an attribute with a specified value on an HTML element.

This method allows you to define an attribute and its value for an HTML element. The attribute will be added to the target HTML node when the returned Consumer is invoked.

If the provided name is null or an empty string, the attribute will not be added.
